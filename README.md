## Setup

```
git clone https://github.com/alisaid-sm/VueExpressAuth.git
```

Masuk ke direktori hasil clone, lalu jalankan

```
npm install
```

Setelah selesai selanjutnnya

```
node index.js
```

Buka Browser ketik localhost:8001
